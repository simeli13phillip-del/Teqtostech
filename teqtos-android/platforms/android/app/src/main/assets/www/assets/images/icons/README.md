## 🎨 Icon Specifications

### Recommended Format: SVG
- **File Type:** SVG (Scalable Vector Graphics)
- **Size:** 24x24px viewBox (will scale automatically)
- **Color:** Use `currentColor` for fill/stroke to inherit text color
- **Style:** Outline or filled, consistent with design system

### Example SVG Structure:
```svg
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg">

</svg>
```

## 📝 Naming Convention

- Use lowercase with hyphens: `home-icon.svg`, `user-profile.svg`
- Keep names descriptive but concise
- Use consistent naming across categories
